# Typecho-hitokoto

### Typecho侧边栏一言插件

> 无论在哪里遇到你，我都会喜欢上你。

调用一言接口 `https://v1.hitokoto.cn/` 返回json

具体接口调用选项参考 [这里](https://hitokoto.cn/api)

json：
````json
// https://v1.hitokoto.cn/

{
  "id": 542,
  "hitokoto": "由变态化身成为变态，也就是真·变态，也就是说，完全变态呢。",
  "type": "a",
  "from": "一拳超人",
  "creator": "没了",
  "created_at": "1472551518"
}

````
